## 🧠 Cách học

1. Đọc tài liệu gốc  
2. Dịch lại sang tiếng Việt  
3. Giải thích theo cách hiểu  
4. Làm lab + demo  
5. Viết writeup

## 🗺️ Roadmap gợi ý

1. Server-side vulnerabilities  
2. Authentication  
3. SQLi → NoSQLi  
4. File Upload → Path Traversal  
5. SSRF → CORS → CSRF  
6. API Testing → GraphQL  
7. Race Conditions  
8. Prototype Pollution  
9. WebSockets  
10. Clickjacking  
11. Web Cache Deception  
12. LLM Attacks
